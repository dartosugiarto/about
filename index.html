Dokumentasi Manual & Sistem PlayBOT

Solusi otomatisasi bisnis PPOB dan Top Up Game via WhatsApp dengan integrasi realtime.

‚ö° Gambaran Umum

PlayBOT adalah bot WhatsApp otomatis yang dirancang untuk menjalankan bisnis PPOB (Payment Point Online Bank) dan Top Up Game tanpa henti. Sistem ini menangani seluruh siklus transaksi mulai dari pengecekan harga, validasi ID game, deposit saldo, hingga pengiriman voucher secara otomatis 24/7.

üöÄ Teknologi Inti

Dibangun dengan stack modern untuk kecepatan dan stabilitas.

Node.js Baileys WA API SQLite

  • Digiflazz: Supplier Utama
  • Duitku: Payment Gateway (QRIS/VA)
  • ApiCheck: Validasi ID Game

üìñ Panduan Pengguna

1. Memulai

Kirim pesan apa saja (contoh: .menu) ke nomor bot. Akun Anda akan otomatis terdaftar di database.

2. Top Up Saldo

üí°
Info: Saldo akan masuk otomatis setelah pembayaran QRIS/VA berhasil dikonfirmasi oleh gateway.
.deposit // Contoh: .deposit 50000

3. Transaksi Produk

Gunakan format berikut untuk membeli produk. Bot akan memvalidasi ID Game (untuk produk game) dan menampilkan nickname untuk konfirmasi.

// Format .buy // Contoh Mobile Legends (Perlu Server ID) .buy ML10 12345678 1234 // Contoh Pulsa (Tanpa Server ID) .buy TSEL10 081234567890

4. Cek Status

  • Cek Saldo: .ceksaldo atau .profil
  • Riwayat Transaksi: .cektrx
  • Cek Harga: .cekstok <kode>

üîß Panduan Admin & Owner

Perintah khusus untuk manajemen sistem. Hanya dapat diakses oleh nomor yang terdaftar sebagai owner/admin.

Keuangan & User
